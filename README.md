## Overview
This is a simple music player application built using Tkinter and pygame in Python. It allows you to add and play MP3 files from a selected folder. You can also control playback, adjust volume, and delete songs from the playlist.

## Dependencies
install using `pip install tk pygame mutagen`
- Tkinter 
- pygame
- mutagen


## Features
- Add songs to the playlist from a selected folder
- Play, pause, stop, go to the next or previous song
- Adjust volume using a slider
- Delete individual songs or clear the entire playlist
- Change themes for a different look and feel

## How to Use
1. Run the script.
2. Click on "File" in the menu and select "Select the song" to add songs to the playlist.
3. Use the control panel to play, pause, stop, or navigate through the playlist.
4. Adjust the volume using the vertical slider.
5. Explore the "Remove songs" menu to delete individual songs or clear the entire playlist.
6. Check out the "Theme" menu to change the appearance.

## Themes
### Neon Theme
- The Neon theme gives the application a vibrant and colorful look.
- To switch to the Neon theme, go to "Theme" in the menu and select "Neon."
